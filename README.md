# OpenSnitch TUI

Terminal UI for [OpenSnitch](https://github.com/evilsocket/opensnitch) application firewall.

## Usage

```bash
sudo opensnitch-tui
```

Requires OpenSnitch daemon installed. Runs as root, auto-manages daemon connection.
